# Pet Routine Tracker - Swift iOS

## Description

Pet Routine Tracker is an iOS application developed using SwiftUI and Core Data. The app helps users manage their pets, organize daily routines, and keep track of pet care activities through a clean and user-friendly interface.

---

## Features

- Add, edit and delete pets
- Add, edit and delete routines
- View pet details
- Pet summary
- Store data locally using Core Data
- Clean and simple SwiftUI interface

---

## Built With

- Swift
- SwiftUI
- Core Data
- Xcode
